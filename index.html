Document Tree Editor

Create quick and easy document trees.
View on GitHub

  • Click an input to modify the name.
  • Hover an input to bring up a branch editing menu.
  • Moved branches are appended to an existing sublist.
  • Deleting a branch will also delete all of its children.
  • Cannot add siblings to or delete the Root dir.
  • Drop a folder to begin editing a file tree. This will erase your current tree! Only works in Chrome.
  • This is written in untranspiled ES6. Turn on experimental JS.
